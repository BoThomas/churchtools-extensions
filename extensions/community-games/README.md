## What is it?

Community Games brings casual gaming to your church community. Create game sessions where church members can join teams and play classic games together ‚Äì perfect for youth groups, community events, or just some friendly competition!

## Features

### üë• Team-Based Gameplay

- **Red Team vs Blue Team**: Community members are assigned to either team at random per game
- **Collective Voting**: Each move is decided by team vote
- **Vote Threshold**: Configurable number of votes needed to make a move
- **Live Updates**: See votes in real-time as team members participate

### üéØ Game Management

- **Create Games**: Set up new game sessions with custom names
- **Game Lobby**: Browse active games and join as a spectator or player
- **Multiple Games**: Run several game sessions simultaneously
- **Game History**: View completed games and their outcomes

## How It Works

### For Organizers

1. **Create a Game**: Go to Settings, name your game, select game type, and set vote threshold
2. **Share**: Tell participants to open the extension and find your game. In the future we could use widgets (in development by ChurchTools)
3. **Play**: Watch as teams vote and compete
4. **Close**: Optionally end the game before it finishes naturally

### For Participants

1. **Open the Lobby**: Find active games listed
2. **Vote on Moves**: Click where you want to play
3. **Win Together**: Celebrate with your team when you get the winning move!

## Game Settings

- **Game Name**: Give your session a recognizable name (e.g., "Youth Night - Round 1")
- **Game Type**: Choose TicTacToe or Connect Four
- **Vote Threshold**: How many team members must vote for the same move before it's played
  - Set to 1 for quick games
  - Higher numbers encourage team decision-making

## Requirements

- ChurchTools instance with Extensions support
- Participants need access to ChurchTools
